# LiveLedger Legal Documents

This folder contains the Terms of Service and Privacy Policy for LiveLedger.

## Deployment

To update the live documents at https://octa-square.github.io/LiveLedger/:

1. Copy `terms-of-service.html` and `privacy-policy.html` to your `octa-square.github.io` repository's `LiveLedger/` folder.
2. Commit and push to deploy.
